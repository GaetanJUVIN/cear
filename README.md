====

CEAR
------------
Creator Empty App Ruby is an internal project to provide a skeleton for ruby app.
Based on Rails structure it will generate a structure like :
```
new_project
  Gemfile
  Rakefile
  app
  config
    application.rb
    database.yaml
    environment.rb
  db
  lib
  log
  tmp
  new_project.rb
```
